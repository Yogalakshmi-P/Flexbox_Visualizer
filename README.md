# 🎯 Flexbox Visualizer

An interactive web app to learn and experiment with **CSS Flexbox** properties in real time.  
Easily adjust properties like `flex-direction`, `justify-content`, `align-items`, and more, and see instant visual feedback.

🔗 **Live Demo:** [Flexbox Visualizer](https://flexbox-visualizer-flex.web.app)

---

## 🛠 Tech Stack & Tools

- **Frontend:** HTML, CSS, JavaScript  
- **Hosting:** Firebase  
- **Tools Used:** VSCode, ChatGPT  

---

## 📌 Features

- 🖱 **Interactive controls** for core Flexbox properties:
  - `flex-direction`
  - `justify-content`
  - `align-items`
  - `align-content`
  - `flex-wrap`
- ⚡ **Instant live preview** of your changes.
- 💻 **Auto-generated CSS code snippet** that updates in real time.
